# Password Generator 🔐

A simple Python project that generates strong and secure random passwords using uppercase letters, lowercase letters, numbers, and special characters. Users can enter the desired password length, and the program instantly creates a secure password. This beginner-friendly project is built using Python and helps improve understanding of randomization and string handling.

## Features
- Generate random passwords
- Custom password length
- Uses letters, numbers, and symbols
- Beginner-friendly

## Technologies Used
- Python
